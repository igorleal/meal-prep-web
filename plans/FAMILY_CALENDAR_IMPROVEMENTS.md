# Family Calendar Improvements

## Overview
Enhance the family calendar feature to improve dietary restrictions handling.

## Tasks

### 1. Load User Dietary Preferences
- [ ] Fetch user's saved dietary restrictions/preferences on calendar load
- [ ] Pre-populate dietary restrictions field with user preferences
- [ ] Ensure preferences sync with user profile settings

### 2. Allow Free Text Dietary Restrictions
- [ ] Implement free text input for additional dietary restrictions (matching meal prep behavior)
- [ ] Support adding multiple custom restrictions
- [ ] Allow users to remove/edit custom restrictions
- [ ] Combine user preferences with custom free text entries

## Acceptance Criteria
- Dietary restrictions auto-load from user preferences
- Users can add free text dietary restrictions just like in meal prep
- Custom restrictions can be added alongside pre-loaded preferences
- UI/UX matches meal prep dietary restrictions input
